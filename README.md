# Secure Bank Database Management System

A secure, scalable banking system built with **Java, JSP/Servlets, and MySQL**.  
It provides safe login, account management, and admin oversight with a user‑friendly interface.

---

## 📖 Overview
The Secure Bank DBMS is designed to streamline digital financial services while ensuring data security and reliability.  
It eliminates manual record‑keeping inefficiencies and offers customers and administrators a robust platform for managing accounts.

---

## ✨ Features
- 🔐 **Secure Login & Session Management** – Reliable authentication with session persistence.  
- 🖥️ **Personalized Dashboard** – Dynamic customer dashboard showing balances and account details.  
- 📂 **Document Upload & Data Integration** – Secure file uploads with backend validation.  
- 🛠️ **Admin Oversight** – Advanced filtering, session management, and activity monitoring.  
- ⚙️ **Scalable Architecture** – DAO pattern with modular servlet layers for maintainability.  
- 🎨 **Accessible UI** – Responsive, branded interface for consistent user experience.  

---

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java (JSP/Servlets), Apache Tomcat  
- **Database:** MySQL with JDBC (MySQL Connector/J)  
- **Architecture:** DAO pattern with connection pooling
